About

This website is intended to describe some of my personal R projects. It may include some posts on other general subjects.

This is my first attempt to use the blogdown package to setup and maintain a website via Hugo and available via GitHub Pages. This site uses the Hyde theme in Hugo.

My first post describes how I set up this site.

Previously I did some posts on WordPress (which I have recreated in this blog) and before that I did some on Blogger.

I am semi-retired. Until 2011 I was the Director of Institutional Research at Yale University. The “semi” part is that I still work for Yale about ten hours a week.
